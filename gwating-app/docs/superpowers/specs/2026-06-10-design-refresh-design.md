# 부팅 디자인 리프레시 — 설계 스펙 (v2: Warm Paper × Electric)

날짜: 2026-06-10
상태: 사용자 승인 완료
시각 기준: `docs/superpowers/specs/2026-06-10-design-refresh-showcase.html`
(브라우저로 열면 6개 화면 시안이 애니메이션과 함께 재생됨 — 구현의 단일 기준)

> v1(A+C 믹스 코랄 시스템)은 이 버전으로 대체됨. 사용자가 쇼케이스 시안을
> 보고 전면 채택을 결정.

## 목표

배포된 부팅 앱의 사용자 페이지 전체를 "프로필 비공개 자동확정 과팅"이라는
제품 본질이 드러나는 디자인 언어로 재구성한다. 디자인 최우선, 코드 정리는
디자인에 필요한 범위로 한정한다.

## 디자인 원칙

1. **Warm Paper** — 순백 금지. 캔버스는 따뜻한 종이색, 흰색은 카드에만
2. **타이포 주도** — 큰 활자가 그래픽. 장식 이모지 제거, 장식은 ✦ 스파크 하나
3. **전기 한 줄기** — 그라디언트는 핵심 액션(매칭 CTA, 내 말풍선, 케미 링)에만
4. **비공개의 미학** — 상대팀 아바타는 프로스트(반투명 ?) 처리로 궁금증을 디자인
5. **감정 곡선 무드** — 탐색 단계는 밝은 페이퍼, 매칭 확정 후엔 다크로 전환

## 디자인 토큰 (`tailwind.config.ts` + `globals.css`)

- **라이트(기본)**
  - `paper` #FAF7F2 (캔버스), 카드 #FFFFFF, `line` rgba(27,25,22,.08)
  - `ink` #1B1916 (본문/잉크 버튼), `muted` #8E887E
  - `electric` 그라디언트 #FF3D5A → #FF7A3D (named 유틸 `bg-electric`)
  - `lilac` #C9B8FF (오로라 블롭 전용)
- **다크(확정 후 무드)**: 배경 #191512(웜 래디얼 #2B2017), 글래스 서피스
  rgba(242,238,231,.06) + 1px rgba 보더 + blur
- **모서리**: 카드 24px / 버튼 18px / 칩 pill
- **그림자**: `shadow-card`(중립 앰비언트), `shadow-glow`(일렉트릭 CTA 전용),
  `shadow-ink`(잉크 버튼)
- **폰트**: Pretendard Variable, 디스플레이는 900 + 타이트 트래킹(-1px대)

## 공용 프리미티브 (`components/ui/`)

| 컴포넌트 | 역할 |
|----------|------|
| `PageShell` | 페이지 래퍼. paper/다크 배경 variant + 스태거 엔트런스 자동 적용 + 프레임 내 높이 처리 |
| `Button` | `ink`(검정) / `electric`(그라디언트+글로우) / `ghost`. 18px, 터치 피드백(active scale 0.97) 내장 |
| `Card` | 24px 흰 카드. `dark`(잉크), `glass`(다크 무드용) variant, `pressable` 옵션 |
| `Chip` | pill 칩, 선택 시 일렉트릭 하이라이트 |
| `BoltLogo` | 볼트 SVG (잉크/그라디언트 variant, 크기 prop) |
| `TabBar` | 글래스 하단 탭바 — 홈/매칭/채팅/마이 4탭, blur + 반투명 |
| `Avatar` | 그라디언트 이니셜 아바타 + `frost` variant(반투명 ? — 비공개 표현) |
| `ChemiRing` | 케미 % 그라디언트 링 (SVG stroke 드로잉 애니메이션) |
| `AuroraBlob` | 블러 처리된 드리프트 블롭 (웰컴/결과/다크 화면 배경) |

별도 Splash 컴포넌트는 만들지 않는다 — 웰컴 화면이 브랜드 모먼트를 흡수.
기존 도메인 컴포넌트(TeamProfileCard, MatchScoreCard, QuizCard 등)는 프리미티브
위에서 리스타일.

## 정보 구조 변경

- **`/` 웰컴** — 비로그인/첫 진입용 브랜드 모먼트. "시작하기" → `/test`,
  "초대 코드로 합류" → `/team/create`
- **`/home` 신설 — 대시보드** — 내 팀 카드(잉크 다크, 멤버 아바타, 분석 진행바,
  매칭 탐색 펄스) + 오늘의 추천 카드(프로스트 아바타, 무드 칩, 케미 %, 비공개
  안내). 테스트/팀 생성 완료 후 랜딩 지점
- **하단 탭바** (v1에서 보류 → 대시보드 구조 채택으로 부활):
  홈 `/home` · 매칭 `/match/result` · 채팅 `/match/chat` · 마이 `/team/demo`.
  탭바는 대시보드 계열 페이지에만 노출(웰컴/테스트 진행 중엔 숨김)
- 라우팅/로직/Supabase 연동은 변경하지 않는다 (페이지 추가 1개 제외)

## 페이지별 적용 + 모션 스펙

모든 화면에 공통: PageShell 스태거 엔트런스, 터치 피드백,
`prefers-reduced-motion` 시 전부 비활성.

| 화면 | 디자인 | 모션 |
|------|--------|------|
| 웰컴 `/` | 페이퍼 + 오로라 블롭, 디스플레이 카피("연애세포, 다시 부팅할 시간."), 잉크 CTA | 블롭 드리프트(무한) + 카피·CTA 순차 페이드업 + 볼트 액센트 |
| 홈 `/home` | 잉크 팀 카드 + 추천 카드 + 글래스 탭바 | 카드 스태거, "매칭 탐색 중" 펄스 닷, 진행바 차오름 |
| 테스트 `/test` | 12분할 세그먼트 진행바, 큰 질문 활자, 선택지 카드(선택 시 그라디언트 보더) | 질문 전환마다 스태거 재생, 선택 팝(scale) |
| 팀 생성 `/team/create` | 폼/칩 새 토큰 통일 | 스태거 + 칩 선택 팝 |
| 팀 프로필 `/team/demo` (마이 탭) | TeamProfileCard 리스타일 + 탭바 | 스태거 |
| 매칭 결과 `/match/result` | ChemiRing 92 + 케미 근거 3줄 + 일렉트릭 CTA | 링 1.6초 드로잉, 근거 순차 등장, 링 플로팅 |
| 일정 `/match/schedule` | 라이트 토큰 적용 | 스태거 |
| 확정 `/match/confirmed` | **다크 무드** — 거대 D-n 그라디언트 숫자, 글래스 정보 카드 | D-n 광택 스윕(무한), 글래스 카드 순차 등장 |
| Q&A `/match/qa` | **다크 무드** — 오늘의 질문 카드(일렉트릭 보더) | 스태거 + 카드 등장 |
| 채팅 `/match/chat` | 페이퍼 톤, 내 말풍선만 일렉트릭, 글래스 헤더/입력바, D-n 필 | 말풍선 순차 등장, 타이핑 인디케이터 도트 |

애니메이션은 전부 CSS keyframes + Tailwind. 라이브러리 추가 없음.

## 데스크톱 모바일 프레임 (`app/layout.tsx`)

- 모바일(<480px) 풀스크린, 데스크톱은 중앙 420px 폰 프레임(라운드+그림자),
  스크롤은 프레임 내부 한정. `min-h-screen` 의존은 PageShell이 흡수

## 작업 순서

1. 파운데이션 — 토큰, 모바일 프레임, 스태거/모션 유틸
2. 프리미티브 — BoltLogo, Button, Card, Chip, Avatar, AuroraBlob, ChemiRing, TabBar, PageShell
3. 웰컴 + 홈 대시보드 (룩 쇼케이스 — 사용자 컨펌 후 진행)
4. 테스트 → 팀 생성/프로필 → 매칭 플로우 5개 마이그레이션 (다크 무드 포함)
5. 미사용 구 스타일 정리
